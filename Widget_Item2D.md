### 2D Overlay Widget
## Item2D
`widget = component.glasses.addItem2D();`

adds a Item Icon to the 2D Overlay

## setItem
* (String) material name
* (Integer) material metaindex

`widget.setItem("minecraft:stone", 6)` 

sets item icon to Andesite Block
